AI Blog
Automated blog system

Tech Used

GitHub Actions, Gemini API, Supabase, Google Blogger API

Purpose

Fully automated AI blogging platform that generates and publishes content.

How It Works

A GitHub workflow runs on a scheduled time interval. The workflow calls an AI API to generate blog content, stores the article in Supabase, and automatically publishes it to Blogger using Google's Blogger API.
